# Authorisation practice

Build authorisation and authentication app to use in future projects.

## Set-up

1. Set up virtual environment

```pip install virtual env```

Go to directory for project and type:

```virtualenv .```

then

```source ./bin/activate```

2. Install django

```pip install django```
